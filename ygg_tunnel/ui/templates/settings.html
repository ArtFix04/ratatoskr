Node Mode

Choose how this node participates in the network. Changes take effect after restart.

💻

Client

Routes your browser traffic through the network via SOCKS5 proxy. Your traffic is anonymised — the exit node's IP is visible to websites.

🔀

Relay

Forwards encrypted traffic between other nodes. You never see the plaintext content — only encrypted bytes pass through. Default for all nodes.

🌐

Exit Node

Connects to the public internet on behalf of other users. Your IP address will be visible to destination websites. Only enable if you understand and accept this responsibility.
